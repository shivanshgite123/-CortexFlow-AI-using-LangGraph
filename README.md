# CortexFlow AI - LangGraph Cognitive Routing and RAG System

## Overview

This project is a multi-agent AI orchestration system built using:

- LangGraph
- LangChain
- OpenAI
- FAISS Vector Database
- RAG

The system performs:

1. Vector-based persona routing
2. Autonomous content generation
3. Prompt injection defense

---

## Features

- Semantic routing using embeddings
- LangGraph workflows
- RAG-based conversational defense
- Structured JSON outputs
- Prompt injection protection
